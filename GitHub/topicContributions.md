# How contribution is recognized on GitHub

## How to show contributions

### Collaborator repos
this is not shown on own profile, but counts in contribution graph

**1) Pin them**
only max 6 on profile

**2) Profile README**
list them in the file

**3) Re-upload zip files from those other repos**
not recommended, lose commit history, might get suspected plagiarism
else keep history, by mirroring, but really better just do 2)

### Forks
shown on profile, but commits not counted in contribution graph

**Merge PR into original repo**
and then let the fork remain, pin to profile

### Private repos
by default not shown on profile, commits also not counted

**Need to go to settings, tick include private contributions**
