# purpose of domain
every website has an ip address, but hard to remember
so domain is alias, easier

different from ip address
ip address changes when change hosting provider ie server, but domain stays same

with domain, can have professional email, ie contact@abelchin.my

# top level domains
| Feature | .com (Generic) | .my (Country Code) |
| ---------- | -------------------- | -------------------------- |
| **Meaning** | commercial (global standard) | for malaysia only |
| **Trust** | Highest globally | High trust within Malaysia, signals you're local |
| **SEO** | Neutral, Good for a global audience | Local SEO boost, google more likely show .my sites to ppl searching from malaysia |
| **Availability** | Hard to find good names (most are taken) | Much better; you're more likely to get your exact name |

# why have different registrars
there's only one registry in malaysia, mynic, manage all .my domains, they prioritize reselling
registrars are the agents, include exabytes, godaddy, sell the domains
registry charge wholesale price, registrars add services so charge more

# transfer domain
if want cheaper renewal fee at different registrar, transfer domain to there
unlock domain in current registrar, get auth code, go to new registrar, select transfer domain, enter code, pay for 1 year

# dns vs registrar
registrar is billing company, authorized sell legal right to domain
dns, domain name system, is internet gps, translate domain into ip address
